package main

 
func main() {
    ch := make(chan int)
    go func() {
        for i := 0; i < 10; i++ {
            ch <- i
        }
    }()
 
    for n := range ch {
        println(n)
    }
	
}

Ответ: программа выводит дедлок, так как ожидает чтения из канала. Можно закрыть канал после отправки в него и тогда программа выполнится успешно.
